# 文件搜索工具 (FileSearch)

一个功能强大、界面美观的文件内容搜索工具，支持实时搜索、进度显示、文件类型筛选等功能。

## ✨ 功能特点

- **实时搜索**：搜索到匹配内容后立即显示结果
- **进度显示**：实时显示搜索进度和文件处理数量
- **文件类型筛选**：支持多种文件类型的筛选，可自定义添加
- **排除目录**：可排除指定目录，如 `.git`、`node_modules` 等
- **支持区分大小写**：可选择是否区分大小写进行搜索
- **结果复制**：右键菜单支持复制文件路径、行号、内容等
- **文件打开**：双击搜索结果可直接打开对应文件
- **现代深色科技风格**：美观的现代深色界面设计

## 📦 安装

### 方法 1：直接使用可执行文件

1. 从 [Releases](https://github.com/lixi5338619/filesearch/releases) 页面下载最新的 `app.exe` 文件
2. 双击运行即可使用，无需安装 Python 环境

### 方法 2：从源码运行

1. 克隆或下载本项目到本地
2. 安装依赖：
   ```bash
   pip install PyQt6
   ```
3. 运行应用：
   ```bash
   python app.py
   ```

## 🚀 使用方法

1. **选择搜索路径**：点击「浏览」按钮选择要搜索的目录
2. **输入关键词**：在关键词输入框中输入要搜索的内容
3. **选择文件类型**：勾选要搜索的文件类型，或在自定义框中输入其他类型（用 `|` 分隔）
4. **设置排除目录**：勾选要排除的目录，或在自定义框中输入其他目录（用 `|` 分隔）
5. **开始搜索**：点击「开始搜索」按钮或按回车键开始搜索
6. **查看结果**：搜索结果会实时显示在下方表格中
7. **复制结果**：右键点击搜索结果可复制相关信息
8. **打开文件**：双击搜索结果可打开对应文件

## 🔧 打包成 EXE

将应用打包成独立的可执行文件：

1. 安装 PyInstaller：
   ```bash
   pip install pyinstaller
   ```

2. 执行打包命令：
   ```bash
   pyinstaller --onefile --windowed app.py
   ```

3. 打包完成后，可执行文件会生成在 `dist` 目录中：
   ```
   dist/app.exe
   ```
